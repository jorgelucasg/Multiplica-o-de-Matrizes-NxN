# Atividade 06/08 - Multiplicação de Matrizes (Python)

## Como usar:

1. Instale o Python (https://python.org) e certifique-se de que o comando `py` funciona no terminal.

2. Coloque os arquivos em uma pasta e abra ela no VS Code.

3. No terminal do VS Code, digite:
   py main.py

4. O programa vai:
   - Ler as matrizes `matriz1.txt` e `matriz2.txt`
   - Verificar se são quadradas
   - Mostrar a multiplicação passo a passo
   - Pedir dois números para testar as funções `somar()` e `multiplicar()`

## Estrutura esperada dos arquivos .txt

Exemplo para `matriz1.txt`:
1 2
3 4

Exemplo para `matriz2.txt`:
5 6
7 8
